# Module 1: Intake & Setup

## Purpose
This module prepares the summarizer for processing the research paper. It normalizes sections, identifies missing or short ones, sets up audience type, and defines chunking rules.

## Functions
- Normalize section headings (default IMRaD if not provided)
- Detect missing or <50-word sections
- Configure audience flags (expert, lay, or both)
- Establish chunk sizes for long texts
